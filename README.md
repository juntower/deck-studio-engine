# deck-studio-engine
根据输入内容进行研究输出可编辑的PPT报告。
咨询级 PPT 全流程生产引擎 | 一句话需求 → 咨询级 .pptx
## 快速开始

直接在对话中说出需求即可触发本技能：

```
"帮我做一份关于 AI 大模型行业格局的 PPT，15 页，面向董事会"
```

引擎会自动执行五阶段流水线，你不需要手动调用任何子步骤。

---

## 输入模板

### 一句话需求（推荐）

```
"帮我做一份 [主题] 的 PPT，[页数] 页，面向 [受众]，风格 [场景]"
```

| 参数  | 必填  | 示例  |
| --- | --- | --- |
| 主题  | ✅   | "AI 大模型行业格局" |
| 页数  | ❌   | "15 页"（默认 10-12 页） |
| 受众  | ❌   | "面向董事会 / 投资人 / 技术团队" |
| 场景  | ❌   | "路演风格 / 咨询报告 / 发布会" |

### 示例输入

```
"AI 创业 BP，12 页，路演融资，A 轮"
"生成一份 2024 中国 AI 政策解读 PPT，10 页"
"把这份研究报告做成 15 页的 PPT，咨询风格"
"技术发布会 PPT，10 页，讲 GPU 算力趋势"
"产品发布 PPT，11 页，面向消费者"
```

## 支持的内容类型

引擎自动识别你的内容并选择最合适的视觉语言：

| 内容类型 | 引擎会做什么 |
| --- | --- |
| **数据/指标** | 选择数据可视化型主角，匹配最佳图表 |
| **流程/架构** | 选择概念示意型主角，生成流程图 |
| **对比/分析** | 选择对比拆分型主角，分栏展示 |
| **趋势/演进** | 选择流程叙事型主角，时间轴布局 |
| **案例/故事** | 选择大图故事型主角，留白+背景图 |
| **金句/观点** | 选择情感共鸣型主角，大字聚焦 |

## 品牌主题切换

引擎会自动推荐最佳主题，你也可以指定：

```
"用路演风格"
"科技感强一点"
```

## 常见问题

### Q: 生成的 PPT 页数不对？

引擎评估内容后自行决定页数。如果想精确控制，在需求中指定："12 页"。

### Q: 生成的风格不是我想要的？

明确指定场景："咨询风格 / 路演风格 / 发布会风格"。也可以在生成后说"换一个更路演风格的模板"。

### Q: 内容数据有误？

引擎会尽量使用最新数据并标注来源。如果发现错误，指出具体位置，引擎会重新 DeepScout 阶段修正。

### Q: 能基于我的文档生成吗？

支持。上传文档（docx/pdf/md），说"根据这个文档做成 PPT"即可。

### Q: 支持批量生成吗？

目前一次处理一份 PPT。如需批量，可分多次调用。

### Q: 生成过程卡住了？

五阶段流水线中每阶段都有质量门禁。如果某阶段数据质量不达标，引擎会自动打回重做，这是正常行为。
