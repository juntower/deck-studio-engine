# 模板蓝图规范（Blueprint）

## 概述

12 种布局模板，每种都有统一的 Blueprint 规范——页面类型、内容槽位、数据绑定规则。

---

## TPL-COVER — 封面页

**用途：** 标题 + 副标题 + 品牌标识
**信息密度：** 低

```
┌──────────────────────────────────┐
│                                  │
│   ████████████████████████████    │
│   ████████████████████████████    │
│                                  │
│       主标题（24-36pt 粗体）       │
│       副标题（16-20pt 中粗）       │
│                                  │
│           日期 | 作者             │
│                                  │
│   ████████████████████████████    │
│   ████████████████████████████    │
│                                  │
└──────────────────────────────────┘
```

**Blueprint:**
```yaml
page:
  template: TPL-COVER
  title: "主标题"
  subtitle: "副标题 (可选)"
  content:
    type: text
    spec:
      date: true
      author: true
  theme: auto
  density: low
```

## TPL-TOC — 目录页

**用途：** 议程 + 章节引导
**信息密度：** 低

```
┌──────────────────────────────────┐
│  目录                              │
│                                  │
│  01 ──── 背景与现状  ──── 3-5页   │
│  02 ──── 核心问题    ──── 3-5页   │
│  03 ──── 解决方案    ──── 3-5页   │
│  04 ──── 实施路径    ──── 2-3页   │
│  05 ──── 价值展望    ──── 1-2页   │
│                                  │
└──────────────────────────────────┘
```

**Blueprint:**
```yaml
page:
  template: TPL-TOC
  title: "目录"
  content:
    type: text
    items:
      - label: "背景与现状"
        pages: "3-5"
      - label: "核心问题"
        pages: "3-5"
  theme: auto
  density: low
```

## TPL-DATA — 数据页

**用途：** 数据图表 + 关键指标标注
**信息密度：** 高

```
┌──────────────────────────────────┐
│  标题                               │
│  ┌────────────────┐   ┌──────────┐ │
│  │                │   │ KPI 1    │ │
│  │    主图表       │   │ 数值      │ │
│  │                │   ├──────────┤ │
│  │                │   │ KPI 2    │ │
│  │                │   │ 数值      │ │
│  └────────────────┘   └──────────┘ │
│  数据来源备注                        │
└──────────────────────────────────┘
```

**Blueprint:**
```yaml
page:
  template: TPL-DATA
  title: "页面标题"
  subtitle: "副标题 (可选)"
  content:
    type: chart
    spec:
      chart_type: bar        # 图表类型
      data_source: "来源"
      data_keys: ["key1", "key2"]
    kpi:
      - label: "关键指标1"
        value: "数值"
      - label: "关键指标2"
        value: "数值"
  theme: auto
  density: high
```

## TPL-COMPARISON — 对比页

**用途：** 左右分栏 + 对比项
**信息密度：** 中

```
┌──────────────────────────────────┐
│  标题                               │
│  ┌────────────┐ ┌────────────┐    │
│  │  方案 A     │ │  方案 B     │    │
│  │            │ │            │    │
│  │ ├ 优势1    │ │ ├ 优势1    │    │
│  │ ├ 优势2    │ │ ├ 优势2    │    │
│  │ └ 不足1    │ │ └ 不足1    │    │
│  └────────────┘ └────────────┘    │
└──────────────────────────────────┘
```

⚠️ **items 格式约束：**
```yaml
# ✅ 正确格式
items:
  - label: "方案A"
    pros: ["优势1", "优势2"]
    cons: ["不足1"]

# ❌ 错误格式（会导致渲染失败）
items: ["方案A", "方案B"]
```
