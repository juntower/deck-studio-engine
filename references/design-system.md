# 设计系统规范

## 雾霁蓝色系设计系统

默认设计语言基于雾霁蓝色系，专业、克制、有温度。

### 语义化配色规则

| 用途 | 默认色 | 说明 |
|------|--------|------|
| 标题文字 | Primary (深蓝) | 保持视觉一致 |
| 正文 | Text (#333) | 高可读性 |
| 数据上升 | #70AD47 (绿) | 正向指标 |
| 数据下降 | #C00000 (红) | 负向指标 |
| 链接/可点击 | #2B5797 | 与主色一致 |
| 引述/强调 | Accent (暖橙) | 不滥用 |

## 布局网格

### 基准网格
- 页面宽高比：16:9 (标准) 或 4:3 (兼容)
- 页边距：上下 40px，左右 48px
- 内边距：标题区下 24px，图表区 16px

### 分栏规则
```
1 栏：  全宽内容（封面、大图、强调）
2 栏：  左 60% + 右 40%（图文混合）
        左 50% + 右 50%（对比页）
3 栏：  等宽 33% + 33% + 33%（数据卡片）
4 栏：  等宽 25% * 4（指标展示）
```

## 图表配色

### 标准图表色板（按使用层级）
```
Chart-1:  #2B5797  深蓝   → 主系列
Chart-2:  #5B9BD5  浅蓝   → 次系列
Chart-3:  #ED7D31  暖橙   → 强调系列
Chart-4:  #70AD47  绿色   → 正向指标
Chart-5:  #FFC000  黄色   → 中性指标
Chart-6:  #C00000  红色   → 负向指标
Chart-7:  #9B59B6  紫色   → 额外维度
Chart-8:  #1ABC9C  青绿   → 额外维度
```

### 图表样式规则
- 柱状图：使用 Chart-1 到 Chart-4 依次分配
- 折线图：Chart-1 主线，Chart-2 辅线，Chart-3 标注
- 饼图/环形图：必须从 Chart-1 开始按顺序取色
- 堆叠图：从 Chart-1 到 Chart-8 依次堆叠
- 散点图：Chart-1 主散点，Chart-3 高亮点

### 视觉比例约束
- 饼图切片 ≤ 6 片（超过则归入"其他"）
- 雷达图维度 ≤ 8（超过则分组）
- 堆叠图系列 ≤ 6（超过则合并）

## 图标与图形

### 图标风格
- 线性图标（line icons）：2px 笔画
- 圆角端点：2px corner radius
- 尺寸标准：24x24px（内联），48x48px（强调）
- 颜色：继承父级文字色或使用 Accent 色

### 装饰元素
- 分隔线：1px, Gray-100, 左右留白 16px
- 项目符号：圆形 ● (2px), 使用主色
- 角标/标签：圆角 4px, 填充 8px, 字号 10pt
- 箭头：→ 或 ▶ (1.2x 字号)

## 信息密度控制

### 逐页密度评估

| 密度等级 | 文字量 | 图表量 | 留白 | 适用页类型 |
|---------|--------|--------|------|-----------|
| 低 | 10-30 字 | 0-1 个小图 | 60%+ | 封面、结尾、强调 |
| 中 | 30-80 字 | 1-2 个图表 | 40% | 概念、对比、流程 |
| 高 | 80-200 字 | 2-4 个图表 | 20% | 数据、表格、路线图 |

### 密度调整规则
- 同一份 PPT 中，密度应从低→高→低波浪式分布
- 开篇（封面+目录）：低密度
- 主体（数据+概念）：中-高密度交替
- 收尾（总结+CTA）：低密度
- 连续不超过 3 页同为高密度（需要插入低密度页"呼吸"）
